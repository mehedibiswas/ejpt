### dnsrecon
- a kali linux builtin-tool
- collect useful info about target
  ### usage
  - dnsrecon -d  hackersploit.org
  - dnsrecon -h
### dnsdumpster.com
- website
- shows dns info
- it also shows a map with hierarchical overviews which is pretty good.
